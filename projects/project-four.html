Modular Cube Interior

Maribor, Slovenia / 2024

The project explores a modular interior system based on a single cube element that defines the spatial organization of the living environment.

Through repetition and flexible arrangement, the module adapts to different functions, creating a compact yet dynamic interior that encourages interaction and everyday use.

CONCEPT

Flexible Modular Living

The modular element functions as seating, storage, shelving and spatial division, allowing the interior to transform according to different needs and activities.

MODULAR SYSTEM
